☁️ Firebase Real-time Sync

ซิงค์ข้อมูลระหว่างหลายเครื่องแบบเรียลไทม์ด้วย Firebase (ฟรี) — ทำครั้งเดียว ใช้ได้ทุกเครื่อง

1
ไปที่ console.firebase.google.com แล้วกด Add project
2
เข้าไปที่ Build → Realtime Database → Create database เลือก Start in test mode → Enable
3
ไปที่ Project Settings ⚙️ → General → Your apps → Add app (เลือก Web </>)
4
คัดลอก firebaseConfig ทั้งก้อนแล้ววางด้านล่าง
⚠️ ข้อมูลจะถูกเก็บใน Firebase ของคุณเอง — ไม่มีใครเข้าถึงได้นอกจากคุณ
🟢
